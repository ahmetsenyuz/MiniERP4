# Mini-ERP Stok ve Sipariş Yönetim Sistemi

Bu proje, küçük ve orta ölçekli ticari işletmelerin ürün yönetimi, stok takibi, tedarik ve sipariş süreçlerini yönetebilecekleri web tabanlı bir Mini-ERP sistemidir.

## Özellikler
- Ürün katalog yönetimi
- Tedarikçi ve depo girişleri yönetimi
- Satış ve sipariş yönetimi
- Anlık stok durumu ve uyarı sistemi
- Yönetici ve operasyon kullanıcı rolleri

## Workflow
- Backlog items → branch-per-item → PR to main
